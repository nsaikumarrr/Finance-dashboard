# Finance Dashboard Backend

## 📌 Project Description
This is a backend application built using Spring Boot to manage financial transactions and provide dashboard insights.

## 🚀 Features
- Add Transactions
- Get All Transactions
- Delete Transactions
- Dashboard Summary (Income, Expense, Count)

## 🛠 Tech Stack
- Java 17
- Spring Boot
- Spring Data JPA
- H2 Database
- Maven

## ▶️ How to Run
1. Open project
2. Run: mvn spring-boot:run
3. Access APIs at: http://localhost:8085

## 📡 API Endpoints
- POST /transactions
- GET /transactions
- DELETE /transactions/{id}
- GET /dashboard
